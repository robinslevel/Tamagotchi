🚕 Taxipoint — Taxi Service Website

A modern, responsive website for a taxi company operating across the Benelux. Designed and developed with a focus on clarity, performance, and user experience.

📅 2025 🌍 Benelux Region 💻 Vanilla HTML/CSS/JS

💡 My Approach

I approached this project from both a design and development perspective. The main challenge was to make a service-based website that feels simple and reliable, while still offering enough functionality for different types of users.

To achieve this, I focused on:

  • Creating a clean and intuitive interface that quickly communicates trust and availability
  • Designing for mobile-first usage, as most customers book taxis on their phones
  • Supporting multiple languages (NL / EN / ES) to reach a broader audience across the Benelux
  • Implementing a dark/light mode to improve usability in different contexts
  • Structuring content to guide users naturally toward booking or contact

⚙️ Key Features

🌍 Multi-language Support

Instant switching between Dutch, English, and Spanish for international users.

📱 Responsive Design

Mobile-first approach with adaptive navigation and touch-friendly interface.

🌙 Theme Toggle

Dark/light mode with system preference detection for comfortable viewing.

📝 Smart Booking Form

Pre-selection options and validation for a smooth booking experience.

🔍 SEO Optimized

Meta tags, Open Graph, and structured data for better discoverability.

♿ Accessibility

ARIA labels and semantic HTML for inclusive user experience.

🧠 What I Learned

This project helped me strengthen my skills in:

  • Building scalable front-end architecture without frameworks
  • Thinking in terms of user flows and conversion instead of just visuals
  • Implementing internationalization (i18n) in a clean and maintainable way
  • Balancing performance, accessibility, and design in a real client project
